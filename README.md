# EnderChest
Bukkit-based Minecraft server plugin. Released under GPLv2.

## Command
- /enderchest
	- Open chest.
	- Aliases: ec, echest
	- Permission: enderchest.use
- /enderchest <page>
	- Open chest with given page.
	- Permission: enderchest.use
- /ecadm

## Permission
- enderchest.use
	- Command usable permission.
	- Default: op
- enderchest.size.<row_int>
- enderchest.extra.<row_int>
